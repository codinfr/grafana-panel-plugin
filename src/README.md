# Bounbu-Panel (Grafana Panel Plugin)

## Overview
This is a custom Grafana **panel plugin** built with React + TypeScript.
It loads inside Grafana and displays:
- Developer name (required)
- Series name + latest numeric value from the query result (bonus)
- Configurable panel options (bonus)

## Requirements
- Docker Desktop
- Node.js
- npm
